//Installation

Vous devez créer une base de données sous le nom de votre choix. en revanche, il vous faudra paramétrer le fichier config.php

Ce fichier de configuration vous permet de paramétrer :
    - le port
    - le nom de la base de données
    - les user / Password pour se connecter à la base de données
    - le hostname
